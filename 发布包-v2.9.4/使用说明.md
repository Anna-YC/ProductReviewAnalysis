# 淘宝评论助手 v2.9.4 传统弹窗版

## 📦 安装步骤

### 1. 解压文件
将 `taobao-review-helper-v2.9.4.zip` 解压到任意文件夹
- 解压后会得到一个名为 `taobao-review-helper-v2.9.4` 的文件夹

### 2. 打开Chrome扩展程序页面
在Chrome浏览器地址栏输入：
```
chrome://extensions/
```
或通过菜单打开：`更多工具 → 扩展程序`

### 3. 开启开发者模式
页面右上角打开 `🔧 开发者模式` 开关

### 4. 删除旧版本（如果已安装）
在扩展列表中找到"淘宝评论助手"，点击"移除"

### 5. 加载扩展
1. 点击 `加载已解压的扩展程序`
2. **选择** `taobao-review-helper-v2.9.4` **文件夹**（包含 manifest.json 的文件夹）
3. 点击"选择文件夹"

### 6. 确认加载成功
看到"淘宝评论助手"出现在扩展列表中即可

---

## 🚀 使用方法

### 启动扩展
- **直接点击浏览器工具栏中的扩展图标**即可打开弹窗

### 📝 提取评论数据
1. 打开**淘宝/天猫/天猫国际**商品详情页
2. 点击浏览器工具栏中的扩展图标
3. 在"📝 评论提取"区域点击 `🚀 开始提取评论`
4. 等待自动跳转和抓取完成
5. 点击 `📥 导出评论` 下载数据

### ❓ 提取问大家数据
1. 打开**淘宝/天猫/天猫国际**商品详情页
2. 点击浏览器工具栏中的扩展图标
3. 在"❓ 问大家"区域点击 `🚀 开始提取问答`
4. 插件会自动：
   - 点击"用户评价"标签
   - 点击"查看全部问答"按钮
   - 进入正式问大家界面
   - 模拟人类操作逐步展开所有回答
   - 滚动加载所有问答
5. 点击 `📥 导出问答` 下载数据

### ⏸ 停止提取
- 点击 `⏸ 停止提取` 按钮
- **已抓取的数据会自动保存**
- 可以立即点击导出按钮下载已抓取的数据

## 🔧 常见问题

**Q: 点击扩展图标没反应？**
A: v2.9.4 已修复，现在点击图标会直接打开弹窗

**Q: 侧边栏不工作？**
A: v2.9.4 恢复为传统弹窗模式，兼容性更好

**Q: 点击停止后还在滚动？**
A: v2.8.7已修复，停止时立即中断滚动并保存数据

**Q: 购买SKU字段值不对？**
A: v2.9.0已优化，现在会提取从【到字符串末尾的完整内容

**Q: 问答进度显示不准确？**
A: v2.9.2已修复，现在显示的是回答数量（展开后），而非问题数量

**Q: 触发了反爬机制？**
A: 已优化防反爬策略，使用随机延迟模拟人类操作

**Q: 评论内容有乱码？**
A: 请使用Excel打开CSV文件时选择UTF-8编码
