# ProductReviewAnalysis
电商产品评论分析系统 - 从用户评价中提取需求，生成营销文案

## 📖 项目简介

基于电商产品评论数据，通过NLP技术分析用户需求，自动生成落地页文案和推销话术。

### 核心功能

| 功能模块 | 说明 |
|---------|------|
| 📊 **数据清洗** | 过滤默认好评、去重、合并初评/追评 |
| 🔑 **关键词提取** | 识别用户最关注的核心词汇 |
| 😊 **情感分析** | 分析用户情绪，识别正面/负面评论 |
| 📈 **方面-观点挖掘** | 按维度分析用户评价（安装/噪音/清洁/吸力等） |
| 🎨 **文案生成** | 自动生成落地页文案框架 |
| 🗣️ **话术生成** | 针对不同场景生成推销话术 |

## 🚀 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

### 运行分析

```bash
# 使用默认数据文件运行
python3 main.py

# 使用自定义数据文件
python3 main.py --data /path/to/your/data.xlsx

# 不生成图表（更快）
python3 main.py --no-charts
```

## 📝 数据格式要求

Excel文件需包含以下列：
| 列名 | 说明 | 必填 |
|------|------|------|
| 序号 | 评论编号 | 是 |
| 用户昵称 | 用户名称 | 是 |
| 评价时间 | 评论日期 | 是 |
| SKU | 产品型号 | 是 |
| 初评内容 | 首次评价 | 是 |
| 追评内容 | 追加评价 | 否 |
