## 🚀 安装方法

### 方法1：拖入安装（推荐）

```
┌─────────────────────────────────────┐
│  Chrome 扩展程序页面          │
│                               │
│  ┌─────────┐   拖入    ┌─────────┐  │
│  │extension  │────────→│  │   │
│  │_ready/   │          │  │   │
│  └─────────┘          │  │   │
│                               │  │   │
└─────────────────────────────────────┘
```

**详细步骤：**

1. **打开Chrome**
   - 启动Chrome浏览器

2. **进入扩展程序页**
   - 地址栏输入：`chrome://extensions/`
   - 或通过菜单：Chrome → 偏好设置 → 扩展程序

3. **启用开发者模式**
   - 在扩展程序页面右上角，打开「开发者模式」开关

4. **拖入插件**
   - 打开Finder，找到项目目录中的 `extension_ready/` 文件夹
   - 将整个 `extension_ready/` 文件夹拖入Chrome扩展程序页面

5. **确认安装**
   - 看到「淘宝评论助手」卡片出现
   - 显示"已启用"状态

### 方法2：加载已解压的扩展

1. **解压扩展**（如果是从ZIP解压的）
   - 找到 `extension/` 或 `extension_ready/` 文件夹

2. **加载扩展**
   - 在chrome://extensions/页面，点击「加载已解压的扩展」
   - 选择扩展文件夹

## ⚠️ 常见问题

### Q1: 拖入后没有反应

**A**: 确保是拖入整个文件夹，不是单个文件
**A**: 确保已启用"开发者模式"
**A**: 尝试刷新扩展程序页面

### Q2: 显示"已停用"或"错误"

**A**: 重新加载扩展
**A**: 检查manifest.json语法
**A**: 查看Chrome扩展程序页面的错误信息

### Q3: 插件图标不显示

**A**: 检查Chrome工具栏右侧（扩展程序图标）
**A**: 点击拼图图标查看已安装的扩展
**A**: 点击扩展的"详细信息"，启用"允许在隐身模式下"

### Q4: 点击后窗口不弹出

**A**: 检查是否启用了弹窗拦截器
**A**: 检查是否是隐身模式
**A**: 尝试刷新商品页面

### Q5: 内容脚本无法提取评论

**A**: 确保已登录淘宝
**A**: 确保在商品详情页（不是搜索结果页）
**A**: 检查网络连接
**A**: 打开开发者工具查看Console日志
