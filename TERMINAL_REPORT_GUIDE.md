# 终端报告 v2.0 使用指南

## 快速开始

```bash
# 自动模式 - 直接查看完整报告
python3 terminal_report_v2.py

# 交互模式 - 选择查看特定维度
python3 terminal_report_v2.py --interactive
```

## 功能特性

- ✅ 自动过滤商家回复内容
- ✅ 移除默认好评
- ✅ 去除重复评论
- ✅ 过滤短评论（<5字符）

### 2. 彩色终端输出
- 🎨 使用 Rich 库美化输出
- 📊 表格化数据展示
- 📈 可视化条形图
- 🎯 颜色分级标记

## 输出说明

### 维度评价颜色标记
- 🟢 **绿色** (≥70%): 用户满意度高
- 🟡 **黄色** (50-70%): 中等评价
- 🔴 **红色** (<50%): 需要改进

### 关键词词频分级
- 🔴 **红色** (≥100次): 高频词
- 🟡 **黄色** (≥50次): 中频词
- ⚪ **白色** (<50次): 低频词

## 常见问题

### Q: 为什么关键词中还有噪音？
A: 停用词配置在 `src/config.py` 中，可以添加更多停用词。

### Q: 如何只分析特定评分的评论？
A: 修改脚本中的数据过滤逻辑，添加评分筛选。

### Q: 可以导出分析结果吗？
A: 使用 `quick_analyze.py` 可生成完整报告文件。
