## 🔧 技术细节

### CSS 变量命名规范

| 类别 | 前缀 | 示例 |
|------|------|------|
| 颜色 | `--color-` | `--color-primary`, `--color-text-primary` |
| 字体 | `--font-` | `--font-size-base`, `--font-weight-medium` |
| 间距 | `--spacing-` | `--spacing-4` (16px) |
| 圆角 | `--radius-` | `--radius-md` (8px) |
| 阴影 | `--shadow-` | `--shadow-md` |
| 动画 | `--duration-`, `--easing-` | `--duration-fast`, `--easing-standard` |

### BEM 命名规范

```
.block {}
.block__element {}
.block--modifier {}
.block__element--modifier {}
```
