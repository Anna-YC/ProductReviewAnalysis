# 淘宝评论助手 v2.8.9

## 📦 安装步骤

### 1. 解压文件
将 `taobao-review-helper-v2.8.9.zip` 解压到任意文件夹

### 2. 打开Chrome扩展程序页面
在Chrome浏览器地址栏输入：
```
chrome://extensions/
```
或通过菜单打开：`更多工具 → 扩展程序`

### 3. 开启开发者模式
页面右上角打开 `🔧 开发者模式` 开关

### 4. 加载扩展
点击 `加载已解压的扩展程序`，选择解压后的文件夹

### 5. 确认加载成功
看到"淘宝评论助手"出现在扩展列表中即可

---

## 🚀 使用方法

### 📝 提取评论数据
1. 打开**淘宝/天猫/天猫国际**商品详情页
2. 点击浏览器工具栏中的扩展图标
3. 在"📝 评论提取"区域点击 `🚀 开始提取评论`
4. 等待自动跳转和抓取完成
5. 点击 `📥 导出评论` 下载数据

### ❓ 提取问大家数据
1. 打开**淘宝/天猫/天猫国际**商品详情页
2. 点击浏览器工具栏中的扩展图标
3. 在"❓ 问大家"区域点击 `🚀 开始提取问答`
4. 插件会自动：
   - 点击"用户评价"标签
   - 点击"查看全部问答"按钮
   - 进入正式问大家界面
   - 模拟人类操作逐步展开所有回答
   - 滚动加载所有问答
5. 点击 `📥 导出问答` 下载数据

### ⏸ 停止提取
- 点击 `⏸ 停止提取` 按钮
- **已抓取的数据会自动保存**
- 可以立即点击导出按钮下载已抓取的数据

## ✨ 功能特点

- ✅ **双模式提取**：评论 + 问大家独立提取
- ✅ **全平台支持**：淘宝、天猫、天猫国际
- ✅ **智能防反爬**：模拟人类操作，随机延迟
- ✅ **智能展开**：自动点击"查看全部回答"
- ✅ **停止可导出**：停止时自动保存已抓取数据
- ✅ **完整数据**：9个字段的完整问答数据
