## ✨ 一致性 优化内容

### 1. 颜色系统统一（使用 Ant Design 标准色）

**优化前**：自定义颜色值，与 Ant Design 不一致

**优化后**：
```css
/* 功能色 - 统一使用 Ant Design 标准色 */
--success: #52c41a;      /* 原 #10b981 */
--warning: #faad14;      /* 原 #f59e0b */
--danger: #ff4d4f;       /* 原 #ef4444 */
--info: #1890ff;         /* 原 #3b82f6 */

/* 新增：配套背景和边框色 */
--success-bg: #f6ffed;
--success-border: #b7eb8f;
--warning-bg: #fffbe6;
--warning-border: #ffe58f;
--danger-bg: #fff1f0;
--danger-border: #ffccc7;
--info-bg: #e6f7ff;
--info-border: #91d5ff;

/* 新增：hover/active 状态色 */
--success-hover: #3da616;
--success-active: #2e8c10;
/* ... 其他色系同样 */
```

### 2. 按钮行为统一

**优化前**：
- 部分按钮缺少 `:active` 状态
- 幽灵按钮没有禁用样式
- 按钮边框处理不一致

**优化后**：
```css
/* 统一禁用状态 */
.btn:disabled,
.btn.disabled {
  opacity: 0.45;
  background: var(--gray-100) !important;
  color: var(--gray-400) !important;
  border-color: var(--border-light) !important;
}

/* 统一 active 状态（所有按钮都有） */
.btn:active:not(:disabled):not(.disabled) {
  transform: scale(0.98);
}

/* 统一边框处理 */
.btn {
  border: 1px solid transparent;  /* 基础边框 */
}
.btn-secondary {
  border-color: var(--border-base);
}
.btn-secondary:hover {
  border-color: var(--primary);  /* hover 变为主题色 */
}
```

### 3. 边框颜色规范

**优化前**：`gray-200`、`gray-300` 混用

**优化后**：统一使用语义化边框变量
```css
/* 边框色 - 统一边框使用规范 */
--border-light: var(--gray-200);  /* 轻边框 */
--border-base: var(--gray-300);   /* 基础边框 */
--border-dark: var(--gray-400);   /* 重边框 */

/* 使用规范 */
.action-card { border: 1px solid var(--border-light); }
.tab-btn { border: 1px solid var(--border-base); }
.btn-secondary { border: 1px solid var(--border-base); }
```

### 4. 组件交互模式统一

**优化前**：交互反馈不一致

**优化后**：
```css
/* 所有可点击元素统一行为 */
.btn,
.tab-btn,
.icon-btn {
  user-select: none;  /* 禁止选中文本 */
  transition: all var(--transition-fast);
}

/* 统一 hover 效果 */
.btn-primary:hover { background: var(--primary-hover); }
.tab-btn:hover { color: var(--primary); border-color: var(--primary); }
.btn-secondary:hover { color: var(--primary); border-color: var(--primary); }

/* 统一 active 效果 */
.btn:active { transform: scale(0.98); }
.tab-btn:active { transform: scale(0.98); }
.icon-btn:active { transform: scale(0.95); }
```

### 5. 过渡时间统一

**优化前**：使用 `var(--transition)`（200ms）

**优化后**：统一使用 `var(--transition-fast)`（150ms）
```css
/* 所有组件统一使用快速过渡 */
transition: all var(--transition-fast);  /* 150ms */
```

### 6. 动画效果精简

**优化前**：`fadeIn` 动画位移 8px，300ms

**优化后**：
```css
@keyframes fadeIn {
  from { opacity: 0; transform: translateY(4px); }  /* 原 8px */
  to { opacity: 1; transform: translateY(0); }
}
```

### 7. 状态指示条优化

**优化前**：颜色使用自定义值

**优化后**：使用统一的功能色
```css
.status-card.ready::before { background: var(--success); }
.status-card.working::before { background: var(--info); }
.status-card.error::before { background: var(--danger); }
.status-card.success::before { background: var(--success); }
```

### 8. Badge 样式统一

**优化前**：激活状态使用 gray-900

**优化后**：激活状态使用主题色
```css
.badge.active {
  background: var(--primary);
  color: white;
  border-color: var(--primary);
}
```

### 9. 代码块颜色统一

**优化前**：硬编码颜色 `#52c41a`

**优化后**：使用 CSS 变量
```css
.code-block.command {
  color: var(--success);
  border-left: 3px solid var(--success);
}
```

### 10. 新增工具类

**新增**：
```css
.disabled {
  pointer-events: none;
  cursor: not-allowed;
}
```
