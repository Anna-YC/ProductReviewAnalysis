## 📦 快速安装

### 方法一：Chrome开发者模式（推荐）

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择本插件目录
5. 完成！

### 方法二：直接拖拽

1. 打开 `chrome://extensions/`
2. 启用「开发者模式」
3. 将插件文件夹拖入浏览器窗口
4. 确认安装

详细说明请查看 [INSTALL.md](INSTALL.md)

## 🚀 使用方法

### 基础流程

```
1. 登录淘宝/天猫
   ↓
2. 打开商品详情页
   ↓
3. 点击插件图标
   ↓
4. 点击"开始爬取评论"
   ↓
5. 等待完成（自动翻页）
   ↓
6. 点击"导出数据"
```

### 支持的平台

- ✅ 淘宝 (taobao.com)
- ✅ 天猫 (tmall.com)
- ✅ 手机端网页版

## ⚙️ 配置选项

编辑 `config.js` 可自定义以下配置：

```javascript
// 抓取配置
MAX_PAGES: 100,              // 最大抓取页数
SCROLL_DELAY: 3000,          // 滚动延迟（毫秒）
PAGE_TURN_DELAY: 2000,       // 翻页延迟（毫秒）

// 导出配置
FILE_PREFIX: '评论数据',     // 文件名前缀
DATE_FORMAT: 'YYYY年MM月DD日' // 日期格式
```

## ❓ 常见问题

### Q1: 插件无法加载？

**A:** 确保：
- 文件夹包含 `manifest.json`
- Chrome 版本 ≥ 88
- 已启用开发者模式

### Q2: 无法提取评论？

**A:** 检查：
- 是否在商品详情页
- 是否已登录淘宝
- 评论区是否已加载

### Q3: 只能抓取少量评论？

**A:**
- 滚动到评论区底部
- 点击"查看全部评价"
- 重新开始抓取

### Q4: 如何与数据分析系统集成？

**A:** 导出的CSV文件可直接用于：
- Python pandas 分析
- Excel 数据透视表
- BI 工具导入
