## 📦 快速安装

### 系统要求

- Chrome 浏览器 114 或更高版本（支持 Side Panel API）
- 或 Edge 浏览器 114 或更高版本

### 安装步骤

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择 `extension` 文件夹
5. 安装完成！

### 使用方法

1. 点击浏览器工具栏上的扩展图标，或按快捷键打开侧边栏
2. 在淘宝/天猫商品详情页打开侧边栏
3. 选择「评论」或「问答」标签页
4. 点击「开始提取」按钮
5. 等待自动翻页完成
6. 导出数据或一键分析

## 🚀 功能说明

### 评论提取

```
1. 打开淘宝/天猫商品详情页
   ↓
2. 打开扩展侧边栏
   ↓
3. 选择「评论」标签页
   ↓
4. 点击「开始提取」按钮
   ↓
5. 自动滚动翻页，实时显示进度
   ↓
6. 导出 CSV 或一键分析
```

### 问大家提取

1. 切换到「问答」标签页
2. 点击「开始提取」按钮
3. 自动提取所有问答数据
4. 导出为 CSV 格式

### 一键分析

提取评论后，点击「一键分析」按钮：
- 数据自动保存到项目目录
- 显示分析命令
- 一键复制命令到剪贴板
- 在终端运行即可生成深度分析报告

## ⚙️ 配置选项

编辑 `config.js` 可自定义以下配置：

```javascript
// 抓取配置
EXTRACTION_TIMEOUT: 600000,  // 提取超时时间（毫秒）
SCROLL_DELAY: 3000,          // 滚动延迟（毫秒）
PAGE_TURN_DELAY: 2000,       // 翻页延迟（毫秒）

// 导出配置
FILE_PREFIX: '评论数据',     // 文件名前缀
CSV_ADD_BOM: true,           // 添加 BOM 头（Excel 兼容）
DATE_FORMAT: 'YYYY年MM月DD日' // 日期格式
```

## ❓ 常见问题

### Q1: 侧边栏无法打开？

**A:** 
- 确保 Chrome 版本 ≥ 114
- 检查是否已启用扩展
- 尝试点击工具栏图标选择「打开侧边栏」

### Q2: 插件无法加载？

**A:** 确保：
- 文件夹包含 `manifest.json`
- Chrome 版本 ≥ 114
- 已启用开发者模式

### Q3: 无法提取评论？

**A:** 检查：
- 是否在商品详情页
- 是否已登录淘宝
- 评论区是否已加载

### Q4: 只能抓取少量评论？

**A:**
- 滚动到评论区底部
- 点击「查看全部评价」
- 重新开始抓取
