# 插件图标

此目录包含插件的图标文件。

## 尺寸要求

- `icon16.png` - 16x16 像素（工具栏）
- `icon48.png` - 48x48 像素（扩展管理页）
- `icon128.png` - 128x128 像素（Chrome Web Store）

## 设计建议

1. 使用简约风格
2. 主色调：橙色 (#ff5000) - 淘宝品牌色
3. 可以包含：购物车、评论气泡、数据图表等元素

## 临时图标

目前可使用纯色占位图标，或从以下资源获取：

- [IconFinder](https://www.iconfinder.com/)
- [Flaticon](https://www.flaticon.com/)
- [IconScout](https://iconscout.com/)

搜索关键词：shopping, review, comment, analysis
