# 淘宝评论助手 - 独立安装指南

## 📦 快速安装

### 方法一：Chrome开发者模式（推荐）

1. **打开Chrome扩展页面**
   ```
   chrome://extensions/
   ```

2. **启用开发者模式**
   - 点击右上角的「开发者模式」开关

3. **加载插件**
   - 点击「加载已解压的扩展程序」
   - 选择本插件文件夹（包含 manifest.json 的目录）

4. **完成**
   - 插件图标出现在浏览器右上角

---

### 方法二：直接拖拽

1. 打开 `chrome://extensions/`
2. 启用「开发者模式」
3. 将本文件夹直接拖入浏览器窗口
4. 确认安装

---

## ⚙️ 配置选项

编辑 `config.js` 文件可自定义以下配置：

### 抓取配置
```javascript
MAX_PAGES: 100,           // 最大抓取页数
SCROLL_DELAY: 3000,       // 滚动延迟（毫秒）
PAGE_TURN_DELAY: 2000,    // 翻页延迟（毫秒）
```

### 导出配置
```javascript
FILE_PREFIX: '评论数据',  // 文件名前缀
DATE_FORMAT: 'YYYY年MM月DD日',  // 日期格式
```

## 🚀 使用方法

### 提取评论数据
1. 登录淘宝/天猫
2. 打开任意商品详情页
3. 点击插件图标
4. 在「📝 评论提取」区域点击「🚀 开始提取评论」
5. 等待完成后点击「📥 导出评论」

### 提取问大家数据
1. 登录淘宝/天猫
2. 打开任意商品详情页
3. 点击插件图标
4. 在「❓ 问大家」区域点击「🚀 开始提取问答」
5. 等待完成后点击「📥 导出问答」

| 权限 | 用途 |
|------|------|
| `activeTab` | 访问当前标签页提取评论 |

## ⚠️ 注意事项

1. **需要在商品详情页使用** - 其他页面无法提取
2. **建议登录后使用** - 可查看更多评论
3. **请勿频繁使用** - 可能触发平台风控
4. **网络不稳定时** - 可能导致抓取中断

---

## 🆘 故障排除

### 插件无法加载
- 确保文件夹包含 `manifest.json`
- 检查Chrome版本是否支持Manifest V3

### 无法提取评论
- 确保在商品详情页
- 检查是否已登录淘宝
- 尝试刷新页面后重试

### 导出失败
- 检查浏览器下载权限
- 尝试在设置中允许下载多个文件
