# 淘宝评论助手 v2.8.2

## 📦 安装步骤

### 1. 解压文件
将 `taobao-review-helper-v2.8.2.zip` 解压到任意文件夹

### 2. 打开Chrome扩展程序页面
在Chrome浏览器地址栏输入：
```
chrome://extensions/
```
或通过菜单打开：`更多工具 → 扩展程序`

### 3. 开启开发者模式
页面右上角打开 `🔧 开发者模式` 开关

### 4. 加载扩展
点击 `加载已解压的扩展程序`，选择解压后的文件夹

### 5. 确认加载成功
看到"淘宝评论助手"出现在扩展列表中即可

---

## 🚀 使用方法

### 📝 提取评论数据
1. 打开**淘宝/天猫/天猫国际**商品详情页
2. 点击浏览器工具栏中的扩展图标
3. 在"📝 评论提取"区域点击 `🚀 开始提取评论`
4. 等待自动跳转和抓取完成
5. 点击 `📥 导出评论` 下载数据

### ❓ 提取问大家数据
1. 打开**淘宝/天猫/天猫国际**商品详情页
2. 点击浏览器工具栏中的扩展图标
3. 在"❓ 问大家"区域点击 `🚀 开始提取问答`
4. 等待自动跳转和抓取完成
5. 点击 `📥 导出问答` 下载数据

## 📊 导出字段说明

### 评论数据（12个字段）

| 序号 | 字段名 | 说明 |
|------|--------|------|
| 1 | ID | 评论唯一标识 |
| 2 | 平台 | 淘宝/天猫/天猫国际 |
| 3 | 用户昵称 | 评论用户名 |
| 4 | 购买SKU | 购买的商品规格 |
| 5 | 首评时间 | 首次评论时间 |
| 6 | 首评内容 | 首次评论内容 |
| 7 | 追评相对时间 | 如"3天后" |
| 8 | 追评内容 | 追加评论内容 |
| 9 | 图片链接 | 评论图片URL |
| 10 | 商家回复 | 商家回复内容 |
| 11 | 点赞数 | 评论点赞数 |
| 12 | 评分 | 商品评分 |

### 问答数据（9个字段）

| 序号 | 字段名 | 说明 |
|------|--------|------|
| 1 | 问题内容 | 用户提出的问题 |
| 2 | 回答者昵称 | 回答用户昵称 |
| 3 | 用户标签 | 如"回头客" |
| 4 | 回答内容 | 回答的具体内容 |
| 5 | 回答时间 | 如"近6天发布" |
| 6 | 购买SKU | 回答者购买的商品规格 |
| 7 | 用户身份 | 如"近期好评"、"半年内已购" |
| 8 | 点赞数 | 回答获得的点赞数 |
| 9 | 评论数 | 回答获得的评论数 |

## ⚠️ 注意事项

1. **页面要求**：必须在商品详情页启动（不要在评价/问答页面启动）
2. **加载完成**：确保页面完全加载后再点击开始
3. **网络稳定**：抓取过程中保持网络稳定
4. **独立操作**：评论和问答需要分别提取，互不影响
5. **停止提取**：如需停止，点击对应的 `⏸ 停止提取` 按钮
6. **兼容性**：适用于Chrome/Edge浏览器（Chromium内核）

## 🔧 常见问题

**Q: 为什么没有显示"成功加载评论区"？**
A: 请确保在商品详情页启动，而不是用户评价页面

**Q: 天猫国际的商品无法抓取？**
A: 请更新到v2.8.2版本，已支持天猫国际各域名

**Q: 问大家数据为空？**
A: 请确认该商品有"问大家"功能，有些商品可能没有问答数据

**Q: 抓取的数据不完整？**
A: 可能是页面加载缓慢，等待时间不足，可以重新抓取

**Q: 评论内容有乱码？**
A: 请使用Excel打开CSV文件时选择UTF-8编码，或直接用记事本/VS Code查看
