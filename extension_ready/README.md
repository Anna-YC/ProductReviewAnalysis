# 🧩 淘宝评论爬取插件 - 安装与使用

## ✅ 插件优势

这是**最优解决方案**，完美解决反爬问题：

| 优势 | 说明 |
|------|------|
| 🔐 **真实浏览器环境** | 无法被检测为爬虫 |
| 🍪 **使用登录态** | 直接用你的登录状态 |
| 👀 **用户可控** | 手动触发和停止 |
| 📊 **实时进度** | 看到爬取进度 |
| 📥 **一键导出** | Excel格式兼容分析系统 |

## 📦 安装步骤

### 方法1：开发者模式安装（推荐）

1. 打开Chrome，输入：`chrome://extensions/`
2. 开启右上角"开发者模式"
3. 将项目中的 `extension/` 文件夹拖入浏览器
4. 完成！

### 方法2：加载已解压的扩展

1. 打开Chrome，输入：`chrome://extensions/`
2. 点击"加载已解压的扩展"
3. 选择 `extension/` 文件夹
4. 完成！

## 🚀 使用方法

### 第1步：打开商品页面

1. 登录淘宝/天猫
2. 浏览器任意商品详情页
   - https://detail.tmall.com/item.htm?id=670501324868
   - 或搜索商品后点击进入

### 第2步：打开插件

点击浏览器右上角插件图标

### 第3步：开始爬取

1. 点击"开始爬取评论"按钮
2. 插件自动：
   - 翻页加载评论
   - 提取评论数据
   - 显示进度条

### 第4步：等待完成

- 查看实时计数
- 达到预期数量后自动停止
- 或点击"停止爬取"

### 第5步：导出数据

1. 点击"导出数据"
2. 自动下载Excel文件
3. 保存到本地

## 📋 提取的数据

插件会自动提取：

| 字段 | 说明 |
|------|------|
| 用户昵称 | 评论用户名 |
| 评论内容 | 评论正文 |
| 评分 | 星级评分 |
| 日期 | 评论时间 |
| SKU | 商品规格 |
| 平台 | tmall/taobao |

## 🎯 导出的数据

点击"导出数据"后，自动下载：
```
评论数据_20260213_HHMMSS.xlsx
```

可以直接用于分析系统：

```bash
python main.py --data ~/Downloads/评论数据_xxx.xlsx
```

## 🔒 隐私说明

### 权限说明

- **activeTab**: 访问当前标签页
- **downloads**: 下载Excel文件
- **storage**: 保存临时数据

### 数据处理

- 所有数据保存在本地
- 不会上传到任何服务器
- 仅用于产品分析

## ⚠️ 注意事项

### 使用限制

1. **需在商品详情页使用** - 其他页面无法提取评论
2. **需登录淘宝** - 未登录只能看到部分评论
3. **请勿过度使用** - 频繁翻页可能触发风控

### 最佳实践

1. **登录后再爬取** - 提高数据完整性
2. **分批爬取** - 单次不超过100页
3. **手动筛选** - 选择需要的评论类型

## 🆕 故障排除

### Q: 插件无法加载

**A**: 确保文件夹结构正确：
```
extension/
├── manifest.json
├── popup.html
├── popup.js
└── content.js
```

### Q: 无法提取评论

**A**: 检查是否在商品详情页，确保登录

### Q: 导出失败

**A**: 检查浏览器下载权限
